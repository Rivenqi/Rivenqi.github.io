Windows下iOS手动抓纯净包方法

特别说明

此方法仅适用于Windows 10系统。
已知不支持iOS系统版本： 16.7.11、17.0~17.4

准备工作：

ios.exe

将tool\go-ios\ios.exe工具拷贝到C:\Windows\System32目录下

wintun

将tool\wintun\bin\amd64\wintun.dll拷贝到C:\Windows\System32目录下

ios驱动

安装tool\i4Tools8_v8.36_Setup_x64.exe

操作步骤

1、查看设备版本及UDID信息

启动控制台输入“ios info”。

其中“ProductVersion”为系统版本信息，“udid”为设备ID信息。

2、挂载镜像（仅iOS 13.1和13.3）

特别说明：挂载成功后不关机可以不再进行以下操作

2.1、将tool\image目录拷贝到任意目录下，记录目录路径，如：D:\image

2.2、执行命令“ios image auto –basedir=”

3、启动隧道（仅iOS 17+需要）

3.1、打开开发者选项

“设置-隐私与安全性-开发者模式”

2.2、启动控制台输入“ios tunnel start –udid= –userspace”。

3、找到应用的BUNDLE_ID

执行“ios apps –list”，第一列为BUNDLE_ID

例如微信“BUNDLE_ID”为“com.tencent.xin”。

3、找到应用的进程ID

根据“BUNDLE_ID”使用“ios ps –apps –nojson”获取所有应用的进程ID（PID），如果没有可能是应用没有启动。

已知微信的“BUNDLE_ID”为““com.tencent.xin”，通过命令的结果可以找到PID为“885”。

4、开始进行抓包

执行“ios pcap –pid=”后开始操作应用，下面以“微信”为例。

“dump-885-1745232619.pcap”为最终生成的报文名，保存在当前命令执行的目录下。

5、结束抓包

使用“Ctrl + C”终止“ios pcap –pid=”命名，“dump-885-1745232619.pcap”报文会从0Kb变成真实大小。
